If I'm not reading manga, watching anime, playing the latest and greatest JRPG or SRPG, I can probably be found working on some sort of software project or hanging around with family. I like working on: emulation, home theatre applications, game development, and scalable network architecture. Some of my projects can be found on GitHub, they include:
  • OpenORPG - A scalable game server for developing networked RPGs that can be played with up to hundreds of players. (https://github.com/hilts-vaughan/OpenORPG)
  • otaku-companion - An ionic framework web-app that runs on mobile devices capable of displaying data from otaku-scraper on a mobile application. (https://github.com/hilts-vaughan/otaku-companion)
  • otaku-scraper - A scraper for the popular MyAnimeList site capable of scraping information related to anime and manga (https://github.com/hilts-vaughan/otaku-scraper)
  • detox - Hacked Final Fantasy XIV and reverse engineered memory. Created a plugin to automate combat and fight for you. (https://www.thebuddyforum.com/rebornbuddy-forum/plugins/154122-team-detox-detox-allinone-plugin.html ; eterna) ... a lot more reverse engineering.
